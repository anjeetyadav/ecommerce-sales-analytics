# E-commerce Sales Analytics Project

## Business Problem

Understanding customer purchasing behavior and sales trends is critical for improving revenue and marketing strategies.
This project analyzes an e-commerce dataset to identify patterns in sales performance, product demand, and customer behavior.

## Objectives

* Analyze sales trends over time
* Identify top-performing products
* Understand customer purchasing patterns
* Generate business insights for revenue growth

## Tools & Technologies

* SQL
* Python (Pandas, NumPy)
* Power BI
* Excel

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. SQL Business Queries
4. Data Visualization
5. Business Insights

## Expected Insights

* Monthly sales growth trends
* Top product categories
* Customer purchasing patterns
* Revenue contribution by category

## Project Structure

data/ – raw dataset
sql/ – SQL queries for analysis
python/ – data analysis scripts
dashboard/ – Power BI dashboard
insights/ – final business insights
