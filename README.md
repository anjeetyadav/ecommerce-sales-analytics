# E-Commerce Sales Analytics

## Project Overview
This project analyzes e-commerce sales data to discover revenue trends, customer behavior, and product performance.

## Tools Used
SQL  
Python  
Power BI  
Excel

## Project Workflow
1. Data Cleaning
2. SQL Data Analysis
3. Python Exploratory Data Analysis
4. Power BI Dashboard
5. Business Insights

## Key Business Questions
• Which products generate the most revenue?  
• Which category has the highest sales?  
• Who are the top customers?  
• What are the monthly sales trends?

## Project Structure
data – raw datasets  
sql – SQL analysis queries  
python – data analysis notebooks  
dashboard – Power BI dashboards  
insights – business findings
